  • التوصيات المدعومة بالذكاء الاصطناعي
  • التسوق المباشر والبث المباشر

🔗 رابط مباشر

⚙️ لوحة الإدارة

🔧 /frontend/admin

إدارة المنصة مع إدارة المستخدمين ومراقبة المحتوى وتكوين النظام.

  • نظرة عامة على المنصة والمقاييس
  • إدارة المستخدمين ومراقبة المحتوى
  • إعدادات النظام والتكوينات
  • إدارة تذاكر الدعم
  • ضوابط علامات الميزات
🔗 رابط مباشر
💡 هل تواجه مشكلة في تحميل الوحدات؟

للحصول على أفضل تجربة، قم بتشغيل هذا المشروع على خادم HTTP محلي لضمان تحميل وحدات ES بشكل صحيح:

# استخدام Python
python -m http.server 8080

# استخدام Node.js
npx http-server -p 8080

# ثم افتح: http://localhost:8080
